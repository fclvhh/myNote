# 作用域链

函数在执行的过程中，先从自己内部找变量

如果找不到，再从创建当前函数所在的**作用域(词法作用域)**去找,以此往上，直到全局作用域

注意找的是变量的此刻的状态

# 柯里化

> 把接收多个参数的函数变成一个单一参数的函数



一个函数返回一个新的函数

```js
function add(a) {
    return function(b) {
    	return a+b
    }
}
// add(3,4)
add(3)(4)
```
