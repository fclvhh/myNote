# js内置对象Math

## 常见用法

| 方法     | 说明                                        |
| -------- | ------------------------------------------- |
| floor()  | 向下取整                                    |
| ceil()   | 向上取整                                    |
| round()  | 四舍五入                                    |
| abs()    | 取绝对值                                    |
| max()    | 最大值                                      |
| min()    | 最小值                                      |
| random() | 返回[0,1)之间的一个伪随机数，包括0，不包括1 |





# 数字

## 数字的写法

| 写法   | 含义     |
| ------ | -------- |
| 0b     | 二进制   |
| 0      | 八进制   |
| 0x     | 十六进制 |
| 后缀 n | 大数     |



## 强制转换

parselnt(string,radix) 与 parseFloat
