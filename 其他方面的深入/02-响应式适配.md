# 媒体查询

## @media



css2时期

1. @media  print
2. @media  screen

```css
/*打印的时候，页面文字是20px*/
@media print {
    body {
        font-size:20px;
    }
}
/*屏幕展示的时候页面文字是16px*/
@media screen {
    body {
        font-size:16px;
    }
}
```
