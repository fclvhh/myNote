# 光标移动

常用操作

| 按键                 | 功能                      |
| -------------------- | ------------------------- |
| ctrl+f               | 向下翻页   pgUp           |
| ctrl+b               | x向上翻页  pgDn           |
| 0/home               | 一行的最前面              |
| $/end                | 一行的最后面              |
| G                    | 文件的末尾                |
| gg                   | 文件的第一行              |
| N [enter]            | 到达文件的任意一行        |
| /word                | 光标向下到文字处          |
| ?word                | 光标向上到文字处          |
| n                    | 配合上面 , 表示重复操作   |
| N                    | 配合上面 , 回到上一次操作 |
| :1,$s/word1/word2/gc | 全文查找并替换            |
|                      |                           |
|                      |                           |



# 删除、复制、粘贴

| dw     | 删除当前单词               |
| yy     | 复制当前行                 |
| y+end  | 当前到行尾                 |
| p      | 粘贴                       |
| J      | 合并                       |
| c      |                            |
| u      | 复原上一个动作             |
| ctrl+r | 重做上一个动作             |
| .      | 重复上一个操作             |
|        |                            |
|        |                            |
